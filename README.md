Society Management System (Spring Boot Backend)

A complete backend system built using Spring Boot, designed to manage Families, Members, and Complaints inside a residential society.
This project demonstrates real-world backend development using layered architecture, relational mappings, validations, exception handling, and RESTful API design.

 Features
1.Family Management

* Create a new family

* Update family details

* Delete a family

* Fetch all families

* Fetch a single family by ID

* Assign flat to a family (with validation)

Member Management

* Add new members to a family

* Update member details

* Delete a member

* Fetch member details

Fetch all members of a family

📋 Complaint Management

* Add or manage complaints per member

* Fetch all complaints for:

* A single member

* A single family

* Delete all complaints of a member

🧱 Tech Stack

* Java 17+

* Spring Boot

* Spring Data JPA / Hibernate

* PostgreSQL / MySQL (any supported database)
